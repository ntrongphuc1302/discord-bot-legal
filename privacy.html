1. Data We Access and Collect

Jun 21, 2026

The Bot interacts with data provided via the Discord API. Depending on the commands you use, this may include:

  • User Identifiers: Discord User IDs and Usernames used to identify and respond to you.
  • Server Identifiers: Server (Guild) IDs, Channel IDs, and Role IDs used for server-specific functionality.
  • Command Content: The content of commands you explicitly send to the Bot to process your requests.
Important: The Bot does not log, read, or store your private personal conversations or messages unrelated to the Bot's specific commands.

2. How We Use the Data

Jun 21, 2026

Any data accessed or temporarily processed is used solely to:

  • Execute and respond to the commands you trigger.
  • Maintain basic functionalities of the Bot within your server.
  • Ensure the security and stability of the Bot (e.g., rate-limiting to prevent spam and abuse).
  • Provide moderation and utility features as designed.

Data is never used for profiling, advertising, or any purpose beyond core Bot functionality.

3. Data Storage and Sharing

Jun 21, 2026

Storage: Data is only stored temporarily in memory or configuration files required for the Bot to function. We do not maintain a permanent database of your personal information. Session data is cleared when the Bot restarts or the process ends.

Sharing: We NEVER sell, trade, rent, or share your data with any third parties. Your data is used exclusively for Bot functionality and is never monetized or shared with advertisers.

4. Data Deletion

Jun 21, 2026

Since data is processed in real-time and minimized, we generally do not hold persistent logs. If you want to ensure any cached data related to your server or user ID is completely removed, you have two options:

  • Kick the Bot from your server — this immediately stops any data access for that server and clears any server-specific data.
  • Contact the developer via Discord to request data removal for your User ID or Server ID.

Data deletion requests are processed within a reasonable timeframe upon verification of your identity.

5. Agreement

Jun 21, 2026

By using the Bot, you consent to the collection and use of information as outlined in this policy. If this policy changes, the "Last updated" date above will be revised and the changes will take effect immediately upon being posted.

We encourage you to periodically review this page for the latest information on our privacy practices. Your continued use of the Bot after any changes to this policy constitutes acceptance of those changes.
